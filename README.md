# AgroFarm Manager

**AgroFarm Manager** — desktop-додаток для автоматизації обліку та планування вирощування сільськогосподарських культур для фермерських господарств та ФОП.

## Основні функції

- **Управління полями**: додавання, редагування та видалення полів, атрибути: назва, площа, тип ґрунту, опис.
- **Каталог культур**: база даних зернових, бобових та олійних культур.
- **Планування посівів**: сезонне планування посадок по полях.
- **Облік витрат**: фіксація витрат на насіння, паливо, добрива тощо.
- **Облік урожайності**: фіксація фактичного врожаю та оцінка якості.
- **Звітність**: формування аналітичних звітів та експорт в Excel.

## Технології та середовище розробки

- Мова програмування: **Python 3.8+**
- GUI: **PyQt6**
- База даних: **SQLite3**
- Додаткові бібліотеки: `pandas`, `openpyxl`, `cx_Freeze` / `PyInstaller`
- IDE: Visual Studio Code / PyCharm
- Контроль версій: Git

## Встановлення та запуск

1. Завантажте `AgroFarmManager.exe`.
2. Скопіюйте файл у будь-яку папку.
3. Двічі клацніть для запуску. Програма автоматично створить базу даних.

## Використання

1. Додайте перше поле через модуль **"Поля"**.
2. Додайте культури через модуль **"Культури"**.
3. Плануйте посіви через модуль **"Планування"**.
4. Фіксуйте витрати через модуль **"Витрати"**.
5. Вносьте врожай через модуль **"Урожайність"**.
6. Формуйте звіти через модуль **"Звіти"** та експортуйте в Excel.

## Примітки

- Програма розрахована на **одного користувача (ФОП / мале господарство)**.
- Мережевий доступ і мобільний додаток наразі не реалізовані.
- Локальна робота без підключення до інтернету.
